# 首页
## 建立知识库缘由
目前前端知识量广，需要学习的东西实在太多。需要建立一个系统的来管理知识，不仅为了记忆和查阅，更多是自己写一遍，再对照经典书籍，就更加能知道自己哪里不足。而且经过一段时间后审查过去所学，不断迭代自己知识的，最终能把知识化为自我，就像JavaScript忍者秘籍作者之一（John Resig）所说
> 像忍者一样，随心所用
